What
====

Save an html-form's user-input to a local file.


How
===

1.) Start server:
python py.py 8080

2.) Fire up browser:
firefox index.html

3.) Click save.


Result:
A file named 'test123456.json' containing the post-data,
should have been saved in this directory, now.
